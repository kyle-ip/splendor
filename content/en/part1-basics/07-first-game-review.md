Played a physical game? Use this checklist before diving into Intermediate strategy.

## Five-Minute Review

1. **Engine status** — How many development cards do you own? Which two colors are deepest?
2. **Noble scan** — Did any noble visit? Was a +3 visit one buy away at any point?
3. **Token discipline** — Did you ever end a turn above 10 tokens? Did a take leave you unable to buy next turn?
4. **Reserve slots** — Were all three reserves on a real buy plan?
5. **Endgame trigger** — Who hit 15 first? Did seat order change who won the round?

## What to Do Next

| If you noticed… | Next step |
|-----------------|-----------|
| Rules still fuzzy | Re-read [Common Mistakes](/learn/basics/part1-basics-06-common-mistakes) |
| Slow buying / token hoarding | Start [Engine Thinking](/learn/intermediate/part2-intermediate-01-engine) |
| Missed nobles | [Noble Planning](/learn/intermediate/part2-intermediate-02-noble-planning) |
| Close loss on points | [Endgame Calculation](/learn/intermediate/part2-intermediate-06-endgame-calc) |

Log one turning point in [replay notes](/tools/replay), then return for Part II when you want a better win rate.

## Key Takeaways

- First games are for **rules fluency**, not perfect strategy
- One honest mistake type per game beats vague “I lost” notes
- Practice tools mirror the same decisions—use them between live games
