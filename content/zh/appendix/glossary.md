## 基础版 — 组件与设置

| 用语 | 含义 |
|------|------|
| **银行** | 桌面中央各色宝石与黄金的指示物堆。 |
| **奖励** | 发展卡右上角的宝石颜色。每 1 点某色奖励，未来购买该色费用减 1，并计入贵族条件。 |
| **发展卡** | 购买后获得声望与奖励的卡牌。三级：一级 40 张、二级 30 张、三级 20 张。 |
| **展示行** | 桌面中央明牌（每级 4 张，牌堆允许时）。 |
| **黄金（万能）** | 野生 **黄金** 指示物；支付时可代替任意一色 1 枚；**只能**通过保留获得（银行有黄金则拿 1 枚）。 |
| **宝石 / 指示物颜色** | 本教程按桌面颜色称呼：**绿、蓝、红、白、黑**，另加万能 **黄金**。说明书亦使用翡翠 / 蓝宝石 / 红宝石 / 钻石 / 缟玛瑙 / 黄金。 |
| **贵族图板** | 明牌；奖励达标时自动来访，**+3 声望**；不可拒绝；每回合最多 **1** 位。 |
| **声望点数** | 发展卡与贵族上的分数；有人 **≥15** 时进入终局轮。 |
| **保留 / 保留卡** | 放入手牌的发展卡（最多 **3** 张）；保留卡不可弃，只能购买。 |

## 基础版 — 规则与流程

| 用语 | 含义 |
|------|------|
| **购买** | 用奖励 + 指示物支付费用；花掉的指示物回银行。 |
| **终局轮** | 有人 ≥15 声望后，**打完当前轮**（每人行动次数相同），再比总分。 |
| **贵族来访** | 在 **回合结束时** 若奖励满足某贵族条件则自动触发；不算一次行动；多位达标时自选一位。 |
| **拿 2 同色** | 仅当拿取时该色银行 **≥4** 枚。 |
| **拿 3 异色** | 三种不同宝石各 1 枚（不含黄金）。 |
| **10 枚上限** | **回合结束时** 指示物（含黄金）合计最多 **10** 枚，多余退回银行。 |
| **平局** | 声望同分时，**购买发展卡更少** 者胜。 |

## 策略用语（本教程）

| 用语 | 含义 |
|------|------|
| **卡牌估值 / 评估** | 判断一张卡是否值得买：分数、引擎、贵族进度与时机，不只看卡面分数。 |
| **阻断 / 否决** | 通过购买、保留或拿宝石拖慢对手；**双人局** 最有效。 |
| **引擎** | 累积的 **奖励** 组合，让你更便宜地连续购卡；一级常建引擎，二三级是「产出」。 |
| **终局计算** | 规划最后几手：谁触发 15、谁还能动、该冲分还是控卡数平局。 |
| **混合路线** | 开局同时铺贵族进度与某一色奖励，介于纯贵族冲刺与单色引擎之间。 |
| **等级经济** | 一 / 二 / 三级卡在费用、分数与购买时机上的差异。 |
| **单色引擎** | 集中某一色奖励，以解锁便宜的高等级购买或完成贵族。 |
| **贵族规划 / 贵族冲刺** | 按贵族条件选路并抢先完成。 |
| **开局** | 前约 5 手：拿币、早期购买、选定贵族 / 单色 / 混合路线。 |
| **读局** | 从可见奖励、指示物、保留数量与展示行推断对手计划。 |
| **保留战术** | 用保留拿金、阻断或锁定下回合购买，而非囤死牌。 |
| **节奏** | 相对对手，把资源换成分数与引擎的速度（加速 vs 蓄力）。 |
| **唯一答案** | 展示行上唯一能完成你贵族或关键购买的那张卡，优先买或留。 |
| **局面演算 / 示例** | 课内短表格，用数字说明推荐走法。 |

## Splendor Duel（对决）

| 用语 | 含义 |
|------|---------|
| **复制颜色卡** | 叠在有奖励的珠宝卡上，本卡奖励算作该色（费用与同色声望均适用）。 |
| **王冠** | 部分珠宝卡印刷；累计第 **3**、第 **6** 个时各拿一张 **皇家卡**；**≥10 王冠** 可获胜。 |
| **黄金（对决）** | 可代替宝石与 **珍珠**；仅能通过「拿 1 黄金 + 保留 1 卡」且棋盘有黄金时获得。 |
| **珠宝卡** | 对决的发展卡（共 67 张），以 **金字塔** 展示（三级 3 / 二级 4 / 一级 5）。 |
| **强制行动** | 每回合必做其一：相邻拿币、金+保留、或购买。 |
| **同色声望胜** | 同一奖励颜色上的卡牌声望 **≥10**（复制色卡随所叠颜色计）。 |
| **可选行动** | 强制行动前：使用特权、和/或补满棋盘（顺序固定）。 |
| **珍珠** | 无对应奖励的指示物；部分费用须珍珠（或黄金万能）。 |
| **特权** | 卷轴（每人最多 3）；交回后可从棋盘拿宝石/珍珠；拿同色 3、2 珍珠或补满棋盘时对手得特权。 |
| **金字塔** | 明牌布局：胜利板块下三级 3、二级 4、一级 5。 |
| **补满棋盘** | 从布袋填空格（中央沿螺旋）；对手获得 1 特权。 |
| **皇家卡** | 第 3 / 6 王冠领取；按卡面结算；不算行动。 |
| **螺旋棋盘** | 指示物在公共棋盘上；拿取须在横、竖或斜的 **不间断** 相邻线上。 |
| **胜利板块** | 三种胜利：**20 声望**、**10 王冠**、**同色 10 声望**；回合结束时达成即 **立即** 结束。 |

完整规则：[第五篇 · 对决](/learn/duel)。

## 扩展（Cities of Splendor）

| 用语 | 含义 |
|------|------|
| **城市板块** | 替代贵族；需声望与奖励组合；取得可结束游戏。 |

## 单人模式（非官方）

| 用语 | 含义 |
|------|------|
| **Automa / 自动机** | 单人对手（骰子或卡牌驱动），从展示行购买或拿牌。 |
| **骰子自动机（模式 2）** | 自动机无法购买时掷骰，可能免费拿一级位或拿金。 |
| **固定资金（模式 1）** | 开局 18 枚后不再从银行拿币；贵族作 **重置次数**。 |
| **重置** | 模式 1 中消耗贵族，弃开展示行并翻 4 张新的一级。 |
| **Solo Automa（模式 3）** | 卡牌驱动的完整双人布局自动机。 |

详见 [单人模式](/reference/solo)。
