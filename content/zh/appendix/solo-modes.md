## 单人模式一：固定资金闯关

**定位**：资源封闭的引擎拼图。开局拿齐宝石后**不再从银行拿宝石**，只靠购买与重置展示行推进。

### 准备

1. **启动资金**：六种宝石指示物（翡翠、蓝宝石、红宝石、钻石、缟玛瑙、**黄金**）**各取 3 枚**，共 **18 枚**，全部归玩家持有。  
2. **发展卡**：仅使用 **一级（Level 1）** 牌堆。洗匀后**正面展示 4 张**，其余为补牌堆。二级、三级本模式**不使用**。  
3. **贵族**：随机取出 **3 张**。本模式中贵族**不计声望**，仅作**重置次数**。

### 回合与行动

无标准「拿宝石 / 保留」。玩家主要进行：

| 行动 | 说明 |
|------|------|
| 购买 | 从展示行买 1 张一级卡；可用奖励与黄金；补牌 |
| 重置 | 重置发展卡 |

**禁止**再从银行拿任何宝石。支付的宝石归还银行后本模式不可再取。

### 过关条件

五种颜色奖励**各自 ≥ 4** 即过关。

### 重置发展卡

1. 消耗 1 张未用贵族（移出，无声望）。  
2. 移出当前 4 张展示卡（建议不放回牌堆）。  
3. 再翻 4 张补满。最多重置 **3** 次。

**练习**：[固定资金闯关](/tools/solo/fixed)（浏览器练习模式）。

## 单人模式二：对抗自动机（骰子版）

**定位**：玩家按标准多人规则行动；自动机按「可买则买最高级，否则掷骰」争夺展示行。

### 准备

二人局展示行 + 3 贵族 + 1 枚六面骰；宝石建议按二人局。自动机开局无有色宝石手牌。

### 回合流程

玩家回合（标准四选一）→ 玩家贵族检查 → 自动机行动 → 自动机贵族检查 → 补牌。

### 自动机行动

1. **优先购买**：三级→一级、同级左→右；仅用其**奖励 + 黄金**判断能否支付；买最高级可买卡。  
2. **否则掷骰**：1–4 免费取一级展示行对应位置卡；5–6 取 1 黄金。

### 终局

任一方 ≥15 声望后按竞速结算；**玩家与自动机均可自动迎贵族**。

**练习**：[骰子自动机](/tools/solo/dice)（浏览器练习模式）。浏览器牌库为 **练习子集**（一级完整 40 张；二级 20/30；三级 15/20）——足以学会模式，并非完整印刷套。

## 单人模式三：Solo Automa（卡牌 Automa）

**定位**：完整二人局桌面 + **Automa 行动牌组**。Automa **不按标准费用购买**，而是「拿指定编号宝石」或「弃置栈底若干筹码后取指定展示卡」。来源：*Splendor Solo Automa*（道具见 `references/solo`）。

![Automa 行动牌 AI-1～AI-9](/assets/solo3/ai-cards-1-9.jpg)

![Automa 行动牌 AI-10～AI-12](/assets/solo3/ai-cards-10-12.jpg)

### 组件（额外）

| 组件 | 数量 / 说明 |
|------|-------------|
| Automa 行动牌 | 共 **12** 张（AI-1～AI-12）；其中 **4** 张右下角带 ★ |
| 代币放置板 | **1** 块，把银行宝石按编号分堆摆放 |
| 牌背 | 印有 Splendor SOLO AUTOMA 的牌背（打印件） |

原规则 PDF：[solo mode 3.pdf](/assets/solo3/rules.pdf)（已放入站点资源）。

### 代币放置板编号

宝石不按「随意堆」而按板子上的编号取用（与牌面数字对应）：

| 编号 | 颜色 |
|------|------|
| **1** | 缟玛瑙（黑） |
| **2** | 翡翠（绿） |
| **3** | 蓝宝石（蓝） |
| **4** | 红宝石（红） |
| **5** | 钻石（白） |

![代币放置板 / 编号图例](/assets/solo3/page5.jpg)

> 本模式 **不使用黄金（万能指示物）** 作为 Automa 拿取目标（牌面数字仅为 1–5 有色堆）。玩家仍可按标准规则拿黄金。

### 行动牌读法

每张牌分 **上 / 中 / 下** 三栏，对应游戏三个阶段（见下「阶段」）：

| 图示 | 含义 |
|------|------|
| 若干带色圆（或数字串如 `5,1,4`） | **拿宝石**：从放置板上取对应编号的各 1 枚，叠入 Automa 的筹码**竖叠**（新拿的叠在上面） |
| 圆箭头 + 数字（如 2 / 3 / 4）+ 等级点 + 四格位标 | **得发展卡**：见下文「得卡」 |
| • / •• / ••• | 目标卡等级：一级 / 二级 / 三级 |
| 四格中一格涂黑 | 该等级展示行从左到右的第 1～4 张 |
| ★ | 高难牌；构筑牌组时特殊处理 |

中文印版上的「黑 / 绿 / 白 / 红 / 蓝」与编号板一致：黑=1，绿=2，蓝=3，红=4，白=5。

### 准备

1. 按**二人局**标准布置发展卡展示行、贵族与宝石；宝石放到**代币放置板**对应编号堆上。  
2. 取出全部 **12** 张 Automa 牌，找出带 ★ 的 **4** 张。  
3. 洗匀这 4 张 ★ 牌，**暗抽 2 张**混入其余 **8** 张无星牌；剩下 2 张 ★ **不看**放回盒中。  
4. 将得到的 **10** 张牌洗匀，牌背朝上放在桌边，形成 Automa 牌库。  
5. Automa 区域：用于竖叠筹码，以及放置其**有声望**的发展卡。

### 阶段与洗牌

Automa 牌库会循环三次，对应 **最多 30 个回合**：

| 阶段 | 回合 | 读牌栏 |
|------|------|--------|
| 第一阶段 | 第 1–10 回合 | **最上栏** |
| 第二阶段 | 第 11–20 回合 | **中栏** |
| 第三阶段 | 第 21–30 回合 | **最下栏** |

- 每一阶段开始前（含开局）：确保牌库已洗匀。  
- 一阶段内：每回合翻 **1** 张；该阶段 10 张用完后，进入下一阶段前**再次洗匀弃牌堆**成新牌库，并改读下一栏。  
- **第 31 回合开始时**：若仍未分出胜负，**玩家失败**（超时）。

### 回合流程

1. **Automa 先手**：翻开牌库顶牌，只执行**当前阶段对应栏**的那一条指示。  
2. **玩家回合**：完全按基础版二人规则行动（含拿宝石、保留、购买、手牌上限等）。  
3. **贵族**：仅**玩家**可迎贵族；**Automa 永不与贵族互动**。  
4. 双方都行动完毕后：若任一方声望 **≥15**，本局结束，**声望高者胜**；否则进入下一回合。

### Automa 行动细则

#### A. 拿宝石（Taking Tokens）

1. 按牌面列出的编号（或中文色标）从放置板各取 **1** 枚。  
2. 全部叠入 Automa 筹码竖叠（可叠在已有筹码之上）。  
3. 若某编号堆已空：**忽略该编号**，其余照拿。  
4. Automa **可以超过 10 枚**筹码（规则允许，虽少见）。

#### B. 得发展卡（Gaining Cards）

1. 按牌面的等级点与四格位标，取展示行中**对应等级、对应位置**的那一张。  
2. 从 Automa 筹码竖叠的**底部**向上，弃置牌面标明数量的筹码（圆箭头旁的数字，如 3 = 弃 3 枚），弃置的筹码回放置板/银行。  
3. **即使筹码不够弃、甚至为 0，仍取得该卡**（Automa 不按卡面真实费用支付）。  
4. 若该卡有声望点数：正面放入 Automa 计分区。  
5. 若该卡 **0 声望**：可直接移出游戏／放回盒中——**不必**为 Automa 追踪奖励（Automa 不靠奖励买卡）。  
6. 展示行按标准从对应牌堆补牌。

若目标位置因耗尽为空：建议取同级最近的一张非空卡，或视为本回合该行动无效只弃牌；开局前约定一种。

### 胜负（模式三）

| 情况 | 结果 |
|------|------|
| 某回合双方行动后，任一方 ≥15 | 比较总分，高者胜 |
| 进入第 31 回合开始时仍未因 ≥15 结束 | **玩家负** |
| 平局 | 规则未写；建议玩家胜，或再比发展卡数量 |

玩家声望 = 发展卡 + 贵族。  
Automa 声望 = 其所保留的**有点数发展卡**之和（无贵族）。

### 与模式二的区别（勿混淆）

| | 模式二（骰子自动机） | 模式三（Solo Automa） |
|--|---------------------|------------------------|
| 对手驱动 | 优先级购买 + 骰子 | 10 张行动牌 × 三阶段 |
| 支付方式 | 奖励+黄金，尽量付清 | 固定弃置栈底 N 枚，可不付清 |
| 是否积累奖励 | 是（买到的卡都留着） | 0 分卡可丢弃，不追踪奖励 |
| 贵族 | 双方都可迎 | **仅玩家** |
| 先手 | 建议玩家先（你此前的模式二） | **Automa 每回合先手** |
| 时限 | 无硬性 30 回合 | **最多 30 回合** |

### 建议裁定（模式三）

| 问题 | 建议 |
|------|------|
| 玩家能否保留 Automa 要拿的位置 | 可以（标准保留） |
| Automa 筹码竖叠「底部」 | 最先放入的那一端；新拿的永远叠在顶 |
| ★ 牌抽到哪 2 张 | 必须暗抽，增加局间难度方差 |
| 中文牌与英文 PDF 图示不一致时 | 以你打印使用的那套图示为准；逻辑相同 |

## 三模式对照

| | 模式一 | 模式二 | 模式三 |
|--|--------|--------|--------|
| 核心 | 封闭资金闯关 | 骰子自动机 | 卡牌 Automa |
| 额外道具 | 无 | 一枚骰子 | Automa 牌 + 代币板 |
| 宝石 | 开局 18 后不再拿 | 玩家标准；机几乎不拿有色 | 玩家标准；机按编号拿 |
| 贵族 | 重置次数 | 双方可迎 | 仅玩家 |
| 胜利 | 五色奖励各 ≥4 | ≥15 竞速 | ≥15 竞速，且 ≤30 回合 |

## 与主规则的关系

- 主线教程默认**多人基础版**。  
- 单人三种写法均在本附录；模式三打印件与规则原文在仓库 `references/solo-automa/`。  
- 模式一练费用规划；模式二练读行与终局；模式三练在 Automa 抢卡压力下的节奏与保留。

| 模式 | 对应 Part 2–3 技能 |
|------|-------------------|
| 一 固定资金 | 宝石经济、一级引擎、机会成本 |
| 二 骰子 | 读桌、否决、终局竞速 |
| 三 卡牌 Automa | 节奏、保留战术、分阶段得分 |
