前 2–3 回合选定的方向，往往决定整局骨架。

## 常见路线

### 贵族冲刺

- 条件：贵族需求与一级展示高度吻合
- 目标：7–9 回合迎来第一位贵族
- 风险：多人同色争夺

### 单色引擎 → 高分卡

- 条件：一级某色卡密集
- 目标：快速堆到该色 3+ 奖励，再连环买二级/三级
- 风险：贵族可能对不上

### 混合路线（最稳）

- 先拿 2–3 张奖励卡，再按补牌决定追贵族或追分
- 适合多数对局，尤其三–四人

## 示例线（T1–T5）

以下是 **模板**，不是剧本——按实际展示调整。

### 贵族冲刺（翡翠–红–玛 的 3/3/3）

| 回合 | 行动 |
|------|------|
| T1 | 拿翡翠 + 红 + 玛（或直接买免费/便宜匹配奖励） |
| T2 | 买展示上最便宜的翡翠或红 |
| T3 | 拿缺色，或买第二张奖励 |
| T4 | 买第三色奖励；被卡住则用黄金桥保留 |
| T5 | 补最后缺口，或拿宝石为下回合迎贵族做准备 |

### 单色蓝引擎

| 回合 | 行动 |
|------|------|
| T1 | 偏蓝拿法（合法且有用时拿 2 蓝） |
| T2–T3 | 买两张便宜 +1 蓝 |
| T4 | 为二级蓝或灵活 2 分卡铺宝石 |
| T5 | 净费用低则买二级；否则第三张蓝奖励 |

### 混合开局

| 回合 | 行动 |
|------|------|
| T1 | 拿三色——对准 **任意** 便宜一级卡 |
| T2 | 买最便宜有用奖励 |
| T3 | 第二张便宜奖励，换 **另一色** |
| T4–T5 | 补牌后：锁定贵族冲刺 **或** 单色深度——别一直犹豫 |

## 局面演算

| 事实 | 数值 |
|------|------|
| 贵族 | 两张都要蓝宝石×3 |
| 一级展示 | 三张便宜蓝奖励、一张零散翡翠 |
| 人数 | 3 |

| 选择 | 结果 |
|------|------|
| 蓝宝石贵族冲刺 | 抢到就快；易被争夺 |
| 单色翡翠 | 弱——场上只有一张翡翠 |
| 混合 | 先拿 1 蓝 + 1 其他便宜奖励，补牌后再定 |

**结论：** 某色在贵族与一级都 **拥挤** 时，只有能赢下争夺才冲刺；否则混合开局，稍后截走路线。

## 动态调整

开局路线 **不是铁律**。第 3 回合补牌完全对不上时，立刻转向——别死磕沉没成本。

## 决策树（简化）

```
贵族吻合展示？ → 是 → 贵族冲刺
               → 否 → 一级某色密集？ → 是 → 引擎路线
                                     → 否 → 混合；先拿 1–2 张奖励
```

完整混合 → 3/3/3 线与中盘转折见 [带注释的双人样例对局](/learn/advanced/part3-advanced-07-sample-game-2p)。
