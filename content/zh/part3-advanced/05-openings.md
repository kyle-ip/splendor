开局 2–3 回合的方向选择，往往决定整局骨架。

## 常见路线

### 贵族冲刺

- 条件：贵族需求与 Level 1 展示高度重合
- 目标：7–9 回合迎第一个贵族
- 风险：多人争抢同色

### 单色引擎 → 高分

- 条件：某色 Level 1 卡密集
- 目标：快速 3+ 同色奖励后连买 Level 2/3
- 风险：贵族可能不吻合

### 混合路线（最稳）

- 先 2–3 张奖励卡，视补牌再决定贵族或冲分
- 适合多数局，尤其 3–4 人

## 局面演算

| 事实 | 数值 |
|------|------|
| 贵族 | 两张都要蓝宝石 ×3 |
| 一级展示 | 三张便宜蓝宝石奖励、一张零散翡翠 |
| 人数 | 3 |

| 选择 | 结果 |
|------|------|
| 蓝宝石贵族冲刺 | 抢得到就快；多人同色易撞车 |
| 单色翡翠 | 弱——展示几乎没有翡翠 |
| 混合 | 先拿 1 蓝 + 1 其他低价奖励，补牌后再定 |

**结论：** 某色在 **贵族与一级展示都扎堆** 时，只有能赢下争夺才冲刺；否则先混合开局，之后再抢路线。

## 动态调整

开局路线 **不是铁律**。若第 3 回合展示行补牌完全偏离，立即切换，沉没成本勿恋战。

## 决策树（简化）

```
贵族与展示重合？ → 是 → 贵族冲刺
                → 否 → 某色 L1 卡多？ → 是 → 引擎路线
                              → 否 → 混合，先拿 1–2 张奖励卡
```
