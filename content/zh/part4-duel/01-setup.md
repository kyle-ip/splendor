## 组件

| 物品 | 数量 / 说明 |
|------|-------------|
| 胜利板块 | 1（标明三种胜利条件） |
| 珠宝卡 | 67 张，分 3 级（一级 / 二级 / 三级牌堆） |
| 布袋 | 1（装已支付/弃回的指示物） |
| 特权卷轴 | 3 |
| 指示物 | 五色宝石各 4、**珍珠 2**、**黄金 3**——共 **25** |
| 棋盘 | 1（印有螺旋的格子） |
| 皇家卡 | 4 |

## 开局步骤（按顺序）

1. **分别洗匀** 三副珠宝牌堆，竖列放置，**一级在最下方**。
2. 在胜利板块下方翻开 **金字塔**：
   - 三级 **3** 张
   - 二级 **4** 张
   - 一级 **5** 张
3. 将棋盘放在金字塔下方。把全部 **25** 个指示物随机放到棋盘上：从 **中央格** 起，沿 **印刷螺旋** 向外填满。
4. 将 **3** 个特权放在棋盘上方。
5. 将 **4** 张皇家卡放在棋盘下方（供之后领取）。
6. **随机** 决定先手。**后手** 拿 **1** 个特权，放在自己面前。

## 桌面布局（心智图）

上 → 下：胜利板块 → 卡牌金字塔（三 / 二 / 一级）→ 指示物棋盘 → 皇家卡。特权在棋盘上方。各玩家已购卡放在自己面前，按奖励颜色整理；保留卡保密。

## 特权提醒（开局相关）

特权稀缺。当棋盘上方供应为空、而你必须拿特权时，从 **对手** 处拿 1 个。若你已持有全部 3 个，则无事发生。
