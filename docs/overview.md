# Overview

Unofficial bilingual learning site for Splendor (EN / 中文).

## Features

- **Layered lessons**: Before you start → Basics → Intermediate → Advanced
- **Interactive tools**: Cost calculator, noble tracker, card valuation, post-game notes
- **Setup wizard**: Player-count–guided table setup (on the intro level page)
- **Solo practice**: Mode 1 (fixed capital) and Mode 2 (dice automa)
- **Appendix**: Rules FAQ, solo mode write-ups, expansion module overview

## Suggested learning path

1. New players: Part 0 + Part 1 (~25 minutes) → play a physical game
2. Stronger win rate: Part 2 + tool practice
3. Competitive play: Part 3 + review notes

## Rules source

Base-game rules in the lessons are aligned with [`references/splendor-rulebook.pdf`](../references/splendor-rulebook.pdf) (Space Cowboys English rulebook).
